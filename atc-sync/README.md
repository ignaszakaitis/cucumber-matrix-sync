# ATC Sync

Syncs the results of the last run and the feature files with the matching
ATCs in the your Matrix project.

## Setup

Create or use existing API token. Please refer: https://docs.matrixreq.com/en/articles/232241-token.
Specify as API_TOKEN in your environmental value
Specify your matrix instance url as SERVER value (e.g https://demo.matrixreq.com)
Specify your matrix project where results should be posted/synched as PROJECT_KEY
Specify folder where your tests are located as XTC_FOLDER_REF (e.g F-XTC-15)
Specify contents field of XTC folder as XTC_CONTENTS_FIELD_NAME
Specify test step column name as STEPS_COMMENT_COLUMN
